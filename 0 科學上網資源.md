# 教程0：开发的前置工作

作为开发者，我们需要顺畅地访问github.com、wikipedia等网站，因此，翻墙是必须的。该教程专为本项目而写，0基础的小白照着做即可完成稳定翻墙（有问题请在Gitter提出）。

## 我们要解决什么问题？

国内屏蔽了大部分开发者必须的服务，如果你希望进行高效地开发以及学习如何开发，访问国外的资源是必须的。而如果你身在国内，由于网络审查的存在，你无法正常访问国外资源。
